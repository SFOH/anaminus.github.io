ROBLOX Client Data

This site contains data about versions of the ROBLOX client, available in multiple formats. Check out the GitHub repository for more information.

This project is in beta. Do not use for any serious work unless you want it to break!

JSON format

  • Format schema
  • Header file
  • Latest API dump
  • Latest ReflectionMetadata file
  • API dumps for specific builds at /rbx/json/api/<version-hash>.json
  • ReflectionMetadata for specific builds at /rbx/json/rmd/<version-hash>.json

Raw format

  • Format schema
  • Header file
  • Latest API dump
  • Latest ReflectionMetadata file
  • API dumps for specific builds at /rbx/raw/api/<version-hash>.txt
  • ReflectionMetadata for specific builds at /rbx/raw/rmd/<version-hash>.xml
